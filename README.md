# Soccer Management System

Starter DDD modular monolith for Phase 2.

## Import into IntelliJ
1. Open IntelliJ IDEA
2. Click Open
3. Select this folder
4. Let Gradle sync
5. Run `SoccerManagementApplication`

## Profiles
- local: H2
- docker: PostgreSQL

## Swagger
- http://localhost:8080/swagger-ui/index.html

## H2 Console
